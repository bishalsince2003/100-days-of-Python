# Day 15 - Coffee Machine ☕

## Concepts Practiced

* Functions
* Dictionaries
* Loops
* Conditional Statements
* Resource Management
* User Input Handling

## Project

Built a Coffee Machine simulation that allows users to order different types of coffee while managing available resources and processing payments.

## Features

* Order espresso, latte, or cappuccino
* Check available resources
* Generate machine reports
* Process coin-based payments
* Calculate and return change
* Deduct ingredients after successful orders
* Turn the machine off using an admin command

## Key Learning

Learned how to break a larger problem into smaller functions, manage program state using dictionaries, and implement real-world logic involving resources, payments, and user interaction.

## Run

```bash
python main.py
```
