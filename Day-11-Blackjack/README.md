# ♠️ Blackjack Game

A Python implementation of the classic Blackjack card game, built as the Day 11 Capstone Project of the **100 Days of Python Bootcamp**.

##  Project Overview

This project simulates a simplified version of Blackjack where the player competes against the computer. The game handles card dealing, score calculation, winner determination, and game flow logic.

##  Features

* Random card generation
* User vs Computer gameplay
* Automatic score calculation
* Blackjack and bust detection
* Winner determination logic
* Replay support
* Console-based interactive experience

##  Concepts Used

* Variables
* Lists
* Dictionaries
* Functions
* Loops (`while`, `for`)
* Conditional Statements (`if`, `elif`, `else`)
* Random Module
* User Input Handling
* Algorithmic Thinking

## 📂 Files

* `main.py` → Initial implementation
* `master.py` → Improved version with enhanced game logic

##  Run

```bash
python main.py
```

or

```bash
python master.py
```

##  Learning Outcome

This project combines most of the concepts learned during the first 10 days of the bootcamp and serves as the first capstone project, focusing on problem-solving, game logic, and program structure.
