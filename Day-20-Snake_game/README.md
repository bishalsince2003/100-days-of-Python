# 🐍 Day 20 - Snake Game

## 📌 Project Overview

This project is part of my **100 Days of Python** challenge. In Day 20, I started building the classic **Snake Game** using Python's Turtle graphics module.

The project focuses on creating the snake, displaying it on the screen, and implementing the basic movement logic.

## 🚀 Features

* Game window created using Turtle Screen
* Snake made from multiple square turtle segments
* Smooth screen animation using `tracer()` and `update()`
* Continuous snake movement
* Object-Oriented Programming using a separate `Snake` class

## 🛠️ Technologies Used

* Python 3
* Turtle Graphics
* Object-Oriented Programming (OOP)

## 📂 Project Structure

```
Day-20-Snake_game/
│── main.py
│── snake.py
```

## 📖 Concepts Learned

* Creating and managing Turtle objects
* Working with lists of objects
* Screen animation with `tracer()` and `update()`
* Creating custom Python classes
* Separating game logic into multiple files

## ▶️ How to Run

```bash
python3 main.py
```
