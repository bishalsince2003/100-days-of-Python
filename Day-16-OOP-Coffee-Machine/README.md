## Concepts Practiced

* Object-Oriented Programming (OOP)
* Classes and Objects
* Methods and Attributes
* Object Interaction
* Modular Programming
* Importing Custom Modules

## Project Overview

Built a fully functional Coffee Machine application using Object-Oriented Programming principles.

The machine can:

* Display available drinks
* Check ingredient availability
* Process coin payments
* Generate reports
* Deduct resources after each purchase
* Shut down using an admin command

## Classes Used

### Menu

Handles drink selection and menu operations.

### CoffeeMaker

Manages ingredients and coffee preparation.

### MoneyMachine

Processes payments and keeps track of profit.

### MenuItem

Represents individual coffee drinks and their requirements.

## Features

* Resource management
* Payment processing
* Profit tracking
* Dynamic menu system
* Modular code structure
* Real-world OOP implementation

## What I Learned

This project introduced me to the practical use of Object-Oriented Programming by making multiple classes work together to simulate a real coffee machine system.
