# 🐢 Day 19 – Turtle Graphics Projects

A collection of interactive Turtle Graphics mini-projects built using Python. These projects focus on event handling, user interaction, and animation while strengthening core Python programming concepts.

---

## 📂 Projects Included

### 🎨 Etch-A-Sketch
Control the turtle using keyboard keys to draw freely on the screen.

**Concepts Practiced**
- Keyboard event listeners
- Turtle movement
- Screen updates

---

### 🏁 Turtle Race
A fun racing game where the user predicts which turtle will win before the race begins.

**Concepts Practiced**
- Random module
- Lists
- Loops
- Turtle Graphics
- User input

---

### ⌨️ Key Movement
Move the turtle in different directions using keyboard controls.

**Concepts Practiced**
- Event handling
- Functions
- Turtle Graphics

---

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics
- Random Module
