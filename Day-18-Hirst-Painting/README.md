# 🎨 Day 18 - Hirst Painting Generator

A Python project inspired by Damien Hirst's famous dot paintings. This program extracts colors from an input image using the **Colorgram** library and recreates a colorful dot painting using Python Turtle graphics.

## 📌 Features

- 🎨 Extracts dominant colors from an image using `colorgram.py`
- 🖌️ Generates a colorful Hirst-style dot painting
- 🐢 Uses Python Turtle graphics for drawing
- 🌈 Randomly selects extracted colors for each dot
- ⚡ Demonstrates loops, functions, lists, tuples, and randomization

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics
- Colorgram.py
- Random Module

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/bishalsince2003/100-days-of-Python.git
```

2. Navigate to the project folder

```bash
cd Day-18-Hirst-Painting
```

3. Create and activate a virtual environment (recommended)

```bash
python3 -m venv .venv
source .venv/bin/activate
```

4. Install the required package

```bash
pip install colorgram.py
```

5. Run the project

```bash
python hirst_painting.py
```
