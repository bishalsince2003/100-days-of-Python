# Day 14 - Higher Lower Game

## Concepts Practiced

- Functions
- Dictionaries
- Loops
- Conditional Statements
- Random Module
- Comparing Data

## Project

Built a Higher Lower game where players compare two accounts and guess which one has more followers.

## Features

- Random account selection
- User guess handling
- Score tracking
- Comparison of follower counts
- Continuous gameplay until a wrong guess

## Key Learning

Learned how to work with complex data structures, compare values from dictionaries, and manage game logic using functions and loops.

## Run

python main.py
