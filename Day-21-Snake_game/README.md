# 🐍 Day 21 - Snake Game (Part 2)

## 📌 Project Overview
This project is part of my **100 Days of Python** challenge. In Day 21, I enhanced the Snake Game by adding interactive gameplay features such as food generation, snake growth, score tracking, and collision detection.

## 🚀 Features
- Keyboard-controlled snake movement
- Random food generation
- Snake grows after eating food
- Live score display
- Wall collision detection
- Self-collision detection
- Game Over message

## 🛠️ Technologies Used
- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)

## 📂 Project Structure

```text
Day-21-Snake_game/
│── main.py
│── snake.py
│── food.py
│── scoreboard.py
```

## 📖 Concepts Learned
- Python OOP
- Creating multiple custom classes
- Keyboard event handling
- Collision detection
- Random module
- Game loop and animation
- Screen updates using `tracer()` and `update()`

## ▶️ How to Run

```bash
python3 main.py
```

## 🎯 Learning Outcome
Built a fully functional Snake Game by combining object-oriented programming concepts with Turtle graphics, including movement, food mechanics, scoring, and game-over conditions.
