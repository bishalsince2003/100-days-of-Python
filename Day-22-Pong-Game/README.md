# 🏓 Day 22 - Pong Game

## 📌 Project Overview
This project is part of my **100 Days of Python** challenge. In Day 22, I built the classic **Pong Game** using Python's Turtle graphics and Object-Oriented Programming. The game features two-player controls, score tracking, collision detection, and progressively increasing ball speed to make the gameplay more challenging.

## 🚀 Features
- Two-player gameplay
- Keyboard-controlled paddles
- Ball movement with smooth animation
- Ball bounces off walls and paddles
- Ball speed increases after every paddle hit
- Score tracking for both players
- Ball resets after a player misses
- Object-Oriented Programming (OOP) implementation

## 🛠️ Technologies Used
- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)

- Dynamic game speed control
