# 🧠 Day 17 – Quiz Game (Object-Oriented Programming)

A command-line Quiz Game built using **Object-Oriented Programming (OOP)** in Python. The application presents True/False questions, evaluates user responses, and keeps track of the score throughout the quiz.

---

## 🚀 Features

- Interactive True/False quiz
- Score tracking after every question
- Object-Oriented design
- Modular code using multiple Python files
- Clean and reusable class structure

---
