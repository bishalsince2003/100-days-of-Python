# Day 9 - Blind Auction

## Concepts Practiced

- Dictionaries
- Loops
- Conditional Statements
- Functions
- Finding Maximum Value

## Project

Built a Blind Auction program that collects bids from multiple participants and determines the highest bidder.

## Features

- Accepts multiple user bids
- Stores bids using a dictionary
- Identifies the highest bid
- Announces the winner automatically
