# Day 12 - Number Guessing Game

## Concepts Practiced
- Functions
- Conditional Statements
- Loops
- Random Module

## Project
Built a Number Guessing Game where the user tries to guess a randomly generated number within a limited number of attempts.

## Features
- Easy and Hard difficulty modes
- Random number generation
- Hint system (Too High / Too Low)
- Attempt tracking

## Run
python main.py
